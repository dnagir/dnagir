Improve HTML Forms

Dec 11th, 2008

Just making a note for myself based on the post Useful resources to improve the look and features of HTML Forms. Thanks to the author for the list:

  • Niceforms - replace the most commonly used form elements with custom designed ones.
  • Spry Validation Form Widget - uses Spry to implement easily validation features in your custom Forms.
  • JavaScriptKit form articles - a lot of ready to use, interesting JavaScript examples.
  • Customselect with icons - degradable select input replacement.
  • JotForm - WYSIWYG form builder.
