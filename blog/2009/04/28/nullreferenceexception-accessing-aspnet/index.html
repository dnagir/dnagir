NullReferenceException Accessing ASP.NET MVC Route Values

Apr 28th, 2009

If by any chance you get NullReferenceException trying to access RouteValues in ASP.NET MVC make sure you have Visual Studio 2008 SP1 installed.
I’d recommend to download the ISO because I had some problems using the web installer.
Have a look at this thread for more details.
